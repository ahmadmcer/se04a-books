PENGANTAR OOP JAVA

OOP (Object Oriented Programming) merupakan suatu metode pemrograman yang berorientasi kepada objek. Tujuan dari OOP dibuat adalah untuk mempermudah pengembangan program dengan cara mengikuti model yang telah ada di kehidupan sehari-hari.

Yang membedakan OOP dengan Pemrograman Prosedural adalah Pemrograman Prosedural Fokus terhadap cara komputasi untuk menyelesaikan suatu tugas. Sedang OOP fokus terhadap objek yang sedang digunakan

TOOLS

  1. JDK (Java Development Kit)

    2. JDK dapat diunduh secara gratis di web oracle

  2. Java IDE

    3. IDE atau Integrated Development Environment digunakan untuk menulis source code. Dan sejumlah IDE yang bisa digunakan untuk keperluan menulis program Java diantaranya : Netbeans, IntelliJ Idea, VS Code, dan Eclipse.

Konsep dasar dari OOP, Yaitu ada Objek, Behavior dan State

  • Object

    • Objek yang biibaratkan seperti "ROBOT" yaitu sebuah tubuh utama

  • Behavior

    • Behavior sebagai perintah dasar untuk menggerakan ROBOT. Perintah dasarnya seperti maju/mundur, belok kanan/kiri, dan lain lain

  • State

    • State jika diibaratkan seperti banyaknya hp atau mp pada ROBOT atau sebuah game yang biasanya menggunakan tipe data integer atau boolean.

    Integer : Jumlah nyawa, mana, shield, peluru

    Boolean : keracunan, menendang bom, melempar bom (aksi yang diakibatkan)

CLASS

prototype/blueprint dr object.

sebuah class mendefinisikan state dan behavior yg dirancang.

kemudian dapat dibuat instance atau object nya.
